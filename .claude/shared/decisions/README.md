# 📚 調査報告書ディレクトリ

このディレクトリには、CCTeamプロジェクトに関する各種調査報告書が保存されています。

## 📝 ファイル命名規則

```
YYYYMMDD_HHMMSS_タイトル調査報告.md
```

- **YYYYMMDD**: 年月日（日本時間）
- **HHMMSS**: 時分秒（日本時間）
- **タイトル**: 調査内容を表す短いタイトル

例：`20250109_193000_エラーループ改善調査報告.md`

## 🔧 新規報告書の作成

```bash
../scripts/create_investigation_report.sh "タイトル" "説明"
```

## 📋 報告書テンプレート

各報告書は以下の構成で作成されます：

1. **概要**: 調査の概要説明
2. **調査目的**: 調査の目的と背景
3. **調査結果**: 詳細な調査内容と発見
4. **提案・改善策**: 短期・長期の対策案
5. **実装計画**: フェーズ分けした実装計画
6. **次のアクション**: 具体的な次のステップ
7. **参考資料**: 関連リンクや文献

- 3ヶ月以上前の報告書は`archive/`サブディレクトリへ移動
